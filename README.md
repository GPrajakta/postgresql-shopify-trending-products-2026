# postgresql-shopify-trending-products-2026
Project: Shopify Trending Products 2026

Tools Used
- PostgreSQL
- SQL

Dataset
Shopify orders dataset (CSV)

Analysis Performed
- Top trending products
- Top Products by Revenue
- Category Revenue Analysis
- Subcategory Sales Performance
- Price Range Popularity
- Products with High Trend but Low Revenue
- Revenue per Unit
- Trend Source Analysis

Key Insights
- Electronics category generates highest revenue
- Products priced $50–$100 sell most units
- Top 10 products contribute 40% of total revenue
